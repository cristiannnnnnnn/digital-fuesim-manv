This software is a digital implementation of the "FüSim MANV"/"FueSim MANV" (Führungssimulation Massenanfall von Verletzen), a German paper-based simulation game for training emergency medical services leadership personnel in managing Mass Casualty Incidents.

For more information look at https://github.com/hpi-sam/digital-fuesim-manv/LICENSE-README.md, for example exceptions that apply to images and datasets and how these two are defined.

You can download a copy of the source code of this software through this link. If this isn't available, you can download it via our github repository.

This software runs with the help of Angular, Node.js, npm Docker and Docker Compose.

Other Third-party-licenses are available to be seen here: THIRD-PARTY-LICENSE-ACKNLODGEMENTS.json We use npm-license-crawler and run `npm run setup` before. For further information look at https://github.com/hpi-sam/digital-fuesim-manv and the package.json and package-lock.json in the root folder and in the subfolders (e.g. frontend, shared or backend).
